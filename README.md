# Pixel Games Studio

This repo is a game studio for building multiple kid-friendly pixel-art web games.

## Goals
- Learn coding step by step
- Build real browser games
- Reuse one shared backend
- Support cloud save and leaderboards
- Prepare safe ad placeholders for future monetization

## Repo Structure

- `apps/` = individual games
- `packages/game-sdk/` = shared backend API helpers
- `packages/ui/` = shared UI components
- `backend/` = shared AWS backend services
- `docs/` = templates and workflow docs

## First Game
- `apps/game-space-runner`

## Phases
1. Core playable game
2. Menus / levels / coins / upgrades
3. Cloud save
4. Leaderboards
5. Ad placeholders + publishing prep
